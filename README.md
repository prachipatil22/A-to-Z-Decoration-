# A to Z Decoration Website 

## Project Overview 
A to Z Decoration is a Web-based platform where users can explore 
and book decoration services for events like weddings,birthday,parties,car,murti,new opening ,house,
and more
## Features 
-Browse different decoration categories
-Easy booking system 
-contact & inquiry option 
-user-friendly interface
## Techologies Used
-Python (Flask)
-HTML,CSS,Javascript
-SQLite database
## Project Structure
-app.py -Main backend file
-templates/ -HTML Pages
-static/-CSS & images
-database-SQLite database
